
#  University Data Dashboard

The interactive dashboard was created in Streamlit, which was first developed in Google Colab, then linked to GitHub, and finally the interactive panel was made in Streamlit based on the information provided by the professor

##  Description
This project is part of the Data Visualization and Dashboard Deployment activity
The goal is to analyze university indicators for retention, satisfaction, and enrollment, and display them in a dynamic dashboard

##  Features

- Filters by year and academic term (Spring/Fall)

- KPI indicators: Applications, Retention Rate, Student Satisfaction

- Visualizations:

  1. Retention Rate (%) trend by year
  2. Average Student Satisfaction (%) by year
  3. Spring vs Fall comparison

- Pivot table with the filtered data

## Technologies Used
- Python
- Pandas
- Matplotlib / Seaborn
- Streamlit

```bash
pip install -r requirements.txt
streamlit run app.py
```
